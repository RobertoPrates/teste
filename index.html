O que é HTML

HTML é HyperText Markup Language, que em português quer dizer linguagem de marcação de hipertexto. Se trata de uma linguagem de marcação que usamos para publicarmos conteúdo (texto, imagem, vídeo, áudio e etc) na Web. Serve para que o navegador o interprete, informando-o como estruturar a página web, e exiba seu conteúdo quando você acessa essa página pela internet.

O que são Clients?

Cliete e um consumidor de infomação da internet, se conectando a internet, se conectando com rede mundial de computadores.

O que são Servers?

Servers?, no bom e velho português chamado de servidor. pode ser um servidor local, que armazena dados de outros computadores dentro de uma rede local em casa mesmo, ou um servidor web, que armazena documentos de varios tipos, via web. O servidores não são computadores comuns, são computadores mais robustos, ou pode ser dizer, com maior capacidade de processamento e armazenamento.

Estruturação básica de um HTML?

A estrutura básica de um documento HTML, ou seja, de uma página da web.
